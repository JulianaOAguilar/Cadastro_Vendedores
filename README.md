# Cadastro de Vendedores

Sistema desenvolvido para cadastro e gerenciamento de vendedores, composto por um **frontend em Angular** e um **backend em Spring Boot**, organizados em um **monorepo**.

## Sobre o projeto

O sistema permite realizar o cadastro de vendedores, armazenando as seguintes informações:

- Nome
- Sexo
- Salário
- Porcentagem de bonificação

Os dados são persistidos em um banco de dados **H2**.

---

# Tecnologias utilizadas 

### Frontend
- Angular
- TypeScript
- HTML (com Bootstrap)

### Backend
- Java
- Spring Boot
- Spring Data JPA
- Maven

### Banco de Dados
- H2 Database


# Funcionalidades

- Cadastro de vendedores
- Listagem de vendedores
- Edição de vendedores
- Exclusão de vendedores
---

# Observações

- O banco H2 é utilizado apenas para desenvolvimento.
- Os dados ficam armazenados localmente.
- Caso o banco esteja em uso por outro processo, encerre a aplicação anterior antes de iniciar novamente.
